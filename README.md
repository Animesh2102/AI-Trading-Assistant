# AI Trading Assistant

A deep learning-based stock prediction system using LSTM and financial data.

## Features
- Stock price prediction using LSTM
- Time-series modeling
- Visualization of predicted vs actual prices

## Tech Stack
- Python
- PyTorch
- yfinance
- NumPy, Pandas

## Project Structure
AI-Trading-Assistant/
├── src/
├── data/
├── models/
├── notebooks/
├── requirements.txt

## Output
Model predicts stock trends and visualizes results.
